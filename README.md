# Cem Besli — Portfolio

Personal portfolio site. Next.js, TypeScript, Tailwind CSS, Framer Motion.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Deploy

1. Push to GitHub (already connected)
2. Import repo at vercel.com/new
3. Deploy — zero config needed
4. Add custom domain: cembesli.com → Settings → Domains → Add → set DNS CNAME to cname.vercel-dns.com
