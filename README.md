# Cem Besli — Portfolio

Personal portfolio. Next.js, TypeScript, Tailwind CSS, Framer Motion. "Precision Light" design.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Deploy

1. Push to GitHub (already connected)
2. Import at vercel.com/new — zero config
3. Add domain: cembesli.com → Settings → Domains → CNAME to cname.vercel-dns.com
